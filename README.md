# Provider/Context demo

- node 22 (latest lts)
- vite 6
- react 18

## Run locally

Download

```
git clone https://github.com/ougarcia/fe_showcase.git
```

Install dependencies

```
npm install
```

Start dev server

```
npm run dev
```

Navigate to [http://localhost:5173/](http://localhost:5173/).
